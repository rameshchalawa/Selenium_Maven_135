Selenium_Maven_135
A Selenium automation framework built with Maven, designed for web application testing.

Overview
This project uses Selenium WebDriver with Java and Maven for dependency management. It is test automation framework for a web application, with a focus on ease of use and scalability.

Project Structure

- src/test/java/pageObjects: Page Object Classes for Framework
- src/test/java/testBase: Framework Baseclass
- src/test/java/testCases: Test Scripts
- src/test/java/utilities: Utilities file like excelutility class for Data Driven testing and ExtentReport class for Report Generation.
- pom.xml: Maven configuration file

Libraries and Frameworks

-Selenium - Web automation
-Maven - Build and package management
-TestNG - Test execution
-Extent Report - Report Generation
-Log4j2 - Test logging

Getting Started

1. Clone the repository: git clone https://github.com/rameshchalawa/Selenium_Maven_135.git
2. Import the project into your IDE as a Maven project
3. Update dependencies in pom.xml
4. Run tests using Maven: mvn test
