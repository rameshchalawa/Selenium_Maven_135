#**Selenium_Maven_135**  

Web automation framework built with Selenium, managed via Maven, executed with TestNG, and enriched with Extent Reports & Log4j2 logging.Implements Page Object Model for efficient test development with a focus on ease of use and scalability.  

**Overview**    
This project uses Web automation framework built with Selenium, managed via Maven, executed with TestNG, and enriched with Extent Reports & Log4j2 logging. Implements Page Object Model for efficient test development with a focus on ease of use and scalability.

**Project Structure**  

- src/test/java/pageObjects: Page Object Classes for Framework
- src/test/java/testBase: Framework Baseclass
- src/test/java/testCases: Test Scripts
- src/test/java/utilities: Utilities file like excelutility class for Data Driven testing and ExtentReport class for Report Generation.
- pom.xml: Maven configuration file  

**Libraries and Frameworks**  
-Selenium - Web automation  
-Maven - Build and package management  
-TestNG - Test execution  
-Extent Report - Report Generation  
-Log4j2 - Test logging  

**Getting Started**  
1. Clone the repository: git clone https://github.com/rameshchalawa/Selenium_Maven_135.git  
2. Import the project into your IDE as a Maven project  
3. Update dependencies in pom.xml  
4. Run tests using Maven: mvn test  

**Key Features- Page Object Factory:**  
-Implements Page Object Model for efficient test development  
-CI/CD Integration: Supports continuous integration
